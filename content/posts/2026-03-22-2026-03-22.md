# 突发！个人微信也能接入 OpenClaw 了

微信终于向「龙虾」敞开大门了。

今天，腾讯正式推出微信「ClawBot」插件。只需在本地安装插件，再用微信扫码登录，就能在个人微信里直接使用电脑中的 OpenClaw。

## 操作步骤：

### 1. 安装 OpenClaw 微信插件并扫码登录

```bash
npx -y @tencent-weixin/openclaw-weixin-cli@latest install
```
